LIGHTNING WEB COMPONENTS (LWC)

Lightning Web Components (LWC) is Salesforce's modern UI framework for building fast, reusable user interfaces.

LWC is built using standard web technologies:

HTML

JavaScript

CSS

Unlike Aura Components, LWC follows modern web standards and provides better performance.

Component Structure

Every LWC contains:

HTML file

JavaScript file

Meta XML file

Optional CSS file

Example

accountCard.html

accountCard.js

accountCard.js-meta.xml
