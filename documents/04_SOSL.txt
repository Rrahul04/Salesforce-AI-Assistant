SOSL

SOSL stands for Salesforce Object Search Language.

Unlike SOQL, SOSL searches across multiple objects simultaneously.

SOQL searches one object.

SOSL searches many objects.

Example

FIND 'Rahul'
IN ALL FIELDS
RETURNING Account(Name),
Contact(Name)

Search Groups

ALL FIELDS

NAME FIELDS

EMAIL FIELDS

PHONE FIELDS

RETURNING Clause

Specifies which objects and fields should be returned.

Example

RETURNING Account(Name),
Lead(Name)
